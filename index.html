High Performance Computing (HPC) at Ghent University

This page serves as an entry point to the open source software tools developed by the HPC team of Ghent University (Belgium).

EasyBuild

EasyBuild is a software build and installation framework written in Python that allows you to install software in a structured, repeatable and robust way.

For more information, see the EasyBuild project page.
